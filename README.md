# Стартовый шаблон для верстки сайтов

## Структура проекта
* `css` -  директория для файлов css:
    + |-- `reset.css`
    + |-- `normalize.css`
    + |-- `main.css`
    + |-- `media.css`

* `fonts` - директория для шрифтов

* `img` - директория для изображений
    + |-- `favicon` - директория для иконок

* `js` - директория для js файлов
    + |-- `libs` - библиотеки js
    + |-- `main.js`
